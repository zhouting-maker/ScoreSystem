## 2. Usage

### 2.1 语句类型判断规则

1. SQL语句中包含 select 或 SELECT 语句类型为 类型1
2. SQL语句中包含 update 或 UPDATE 语句类型为 类型2
3. SQL语句中包含 delete 或 DELETE 语句类型为 类型3
4. SQL语句中包含 insert 或 INSERT 语句类型为 类型4
5. SQL语句中包含 create 或 CREATE 语句类型为 类型5
6. SQL语句中包含 drop 语句类型为 类型6
7. SQL语句中包含 alter 语句类型为 类型7
8. SQL语句中包含 truncate 语句类型为 类型8
9. SQL语句中包含 replace 语句类型为 类型9
10. SQL语句中包含 merge 语句类型为 类型10
11. SQL语句中包含 call 语句类型为 类型11
12. SQL语句中包含 execute 语句类型为 类型12
13. SQL语句中包含 prepare 语句类型为 类型13
14. SQL语句判断过程中，大小写不敏感
